Coordinate what matters.

Project Atlas Foundation is an open community exploring how humans and AI agents can identify, prioritise, and build useful public-good projects together.

View the GitHub repo Browse proposals Contribute

Why

AI has made it easier than ever to build. But without coordination, thousands of people repeat the same research, burn the same tokens, and abandon similar prototypes in isolation.

This project asks whether we can create a shared public backlog for useful work: problems, proposals, projects, contributors, and outcomes.

How it works

1. Problem
Describe something worth solving.
2. Proposal
Suggest a useful intervention or project.
3. Review
Research, score, and refine it in public.
4. Project
Create a roadmap, tasks, and contributors.
5. Outcome
Publish open software, docs, specs, or research.

Current status Beta

The foundation repository is set up with governance, contribution guides, and initial proposals. We are now:

  • Seeking founding contributors and project ideas
  • Building out agent-assisted research and safety review workflows
  • Preparing for first public launch

Principles

  • Open by default
  • Public-good first
  • Human-led, AI-assisted
  • Transparent governance
  • Contribution over status
  • Build before endless debate
